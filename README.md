# Notebooks for post-analysis of `redsun-mimir` acquisitions

A series of notebooks for post-processing analysis related to [`redsun-mimir`](https://github.com/redsun-acquisition/redsun-mimir).

> [!WARNING]
> Work in progress. Use at your own risk.

## Installation

> [!NOTE]
> Requires [`uv`](https://docs.astral.sh/uv/)

Clone the repository:

```bash
git clone https://github.com/redsun-acquisition/redsun-mimir
```

Create a virtual environment (requires `python>=3.13`)

```bash
uv venv --python 3.13

# in Windows
.venv\Scripts\activate
```

Install dependencies via:

```bash
uv sync
```
